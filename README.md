# Expense Tracker

A simple Expense Tracker web application built using HTML, CSS, and JavaScript. It helps users record income and expenses, calculate the current balance, and manage transactions easily.

## Live Demo
https://karrisribhagyadhaneswarareddy-maker.github.io/Expense-Tracker/

## Features
- Add income and expense transactions
- Display total balance
- Show total income and expenses
- Delete transactions
- Simple and responsive user interface
- Data saved using Local Storage

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Use
1. Enter a description.
2. Enter the amount.
3. Select Income or Expense.
4. Click **Add Transaction**.
5. View your updated balance and transaction history.
